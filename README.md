# Corporate Expenses Dashboard

An interactive, VBA-powered Excel dashboard designed to provide comprehensive insights into corporate spending, budget allocation, and departmental expenses. This tool automates the financial tracking process, allowing stakeholders to easily monitor financial health and make data-driven decisions.

## 📊 Project Overview

Managing corporate expenses efficiently requires clear visibility into where funds are allocated versus where they are actually spent. This dashboard solves that problem by transforming raw transaction data into a clean, intuitive interface. By leveraging VBA macros and dynamic Excel features, it automates data updates and provides a real-time snapshot of the company's financial status.

## 🚀 Key Performance Indicators (KPIs)

At the top of the dashboard, users can immediately gauge the overall financial standing through several critical metrics:
*   **Total Expenses:** The aggregate amount of corporate spend (e.g., $112K).
*   **Budget Allocated & Remaining:** A direct comparison of the total approved budget (e.g., $192K) versus what is left in the reserves ($80K), helping prevent departmental overspending.
*   **Transactions & Avg Expense:** Tracks the volume of recorded transactions and the average cost per transaction to identify spending behaviors.
*   **Approval Rate:** Monitors the operational compliance of expense requests (currently tracking at 100%).

## 📈 Detailed Visual Analytics

The dashboard breaks down the high-level KPIs into granular visual insights:
*   **Expense by Department:** A bar chart comparing how much capital is being utilized by specific business units (e.g., Finance vs. other departments).
*   **Budget vs. Actual:** A clustered column chart providing a side-by-side variance analysis of allocated funds against actual expenditures.
*   **Monthly Trend:** A line graph tracking the velocity of spending over the fiscal year, making it easy to spot seasonal spikes or anomalies in specific months.
*   **Expense by Category:** A donut chart segmenting the expenses into specific operational buckets, such as Office Supplies.
*   **Payment Method:** A horizontal chart detailing the preferred transaction channels, such as UPI.

## 🎛️ Interactive Elements

The right-hand panel features interactive slicers that allow users to slice and dice the data dynamically without altering the raw dataset:
*   **Time Period Filtering:** Filter the entire dashboard by specific months (e.g., isolating August data).
*   **Departmental Filtering:** Isolate the metrics and charts to view performance for a single department like Finance, HR, IT, Marketing, Operations, or Sales.

## 🛠️ Technical Stack
*   **Microsoft Excel:** Advanced Data Modeling, PivotTables, and dynamic charting.
*   **VBA (Visual Basic for Applications):** Backend automation for data processing and seamless dashboard interactivity.

## ⚙️ How to Use This Dashboard
1. Clone or download this repository to your local machine.
2. Open the `.xlsm` file using Microsoft Excel.
3. **Important:** Click **"Enable Content"** or **"Enable Macros"** if a security warning appears at the top of your screen. The VBA automation will not run without this step.
4. Use the slicers on the right side to filter the dashboard by Month or Department and watch the KPIs and charts update automatically.
